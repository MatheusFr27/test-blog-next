## Passos

### Dependências

```bash
npm i
```

### Configure o Env

Copie o ```.env.example``` em um novo .env e coloque os dados do banco de dados e as credencias do google para obter a autenticação através do Google.

### Inicie o servidor 

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
